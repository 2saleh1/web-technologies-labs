# CS 471 Web Technologies Labs

This repository contains laboratory exercises and projects for CS 471 Web Technologies course.

## Labs

### 1. Wireshark Lab
Network protocol analysis using Wireshark packet capture tool.
- TCP/IP protocol inspection
- Three-way handshake analysis
- Data transfer monitoring

**Location**: `wireshark-lab/`

### 2. Django Web Application
Backend web development project using Python Django framework.
- Web application architecture
- Server-side development
- Database integration

**Location**: `django-webapp/`

## Getting Started

Each lab directory contains its own README with specific instructions and materials.

## Purpose

These labs provide hands-on experience with:
- Network protocols and packet analysis
- Web application development
- Backend frameworks and architecture
- Real-world web technology implementations

# Agent Session Overview

## Purpose

This is a GitHub Copilot Agent session working on the **web-technologies-labs** repository. The repository contains lab assignments and projects for **CS 471 Web Technologies** course.

## Repository Context

### What is This Repository?

This repository serves as a collection of web technology lab exercises and projects for the CS 471 course. It demonstrates practical implementations and explorations of various web technologies and networking concepts.

### Current Contents

The repository currently contains:

1. **wireshark-lab/** - Network protocol analysis lab
   - Contains Wireshark packet capture analysis
   - Includes documentation and screenshots of:
     - TCP three-way handshake
     - Data transfer analysis
     - Network packet inspection

2. **django-webapp/** - Django web application project
   - A Django-based web application project
   - Demonstrates backend web development concepts
   - Uses Python Django framework
